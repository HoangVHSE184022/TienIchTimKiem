# React Native  

### Môi trường dự án: 
Node.js v20.17.0  

### Các bước cài đặt:  
1. Clone code  
2. Cài đặt thư viện: npm i  
3. Update file .env (nếu có)  
4. Chạy dự án: npm start || npm run dev  
5. Chọn thiết bị để build app (a cho android và quét mã để sử dụng cho ios)
6. Đóng dự án: Ctrl + C  
* Để refresh app: nhấn r
